# Quick Start Guide

## Getting Started

Welcome to the Markdown to PDF Converter! This guide will help you get up and running in minutes.

### Installation

1. Clone the repository
2. Run `uv sync`
3. Start converting!

### Your First Conversion

```bash
uv run md2pdf document.md
```

That's it! Your PDF will be created in the same directory.

### Features

| Feature | Description |
|---------|-------------|

| Beautiful | Professional-looking output |
